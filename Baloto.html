Agrupando variables

Una Particular relacion se encuentra entre la balota n 3 y balota n4 ya que pareciere indicar:

  1. Correlacion directa entre balota N 3 y N2
  2. COrrelacion inversa entre balota N 3 y N4
par(mfrow = c(1, 3))
plot(baloto$Balota_3, baloto$Balota_2, main = "Relacion entre Balota N3 y Balota N2", cex.main = 0.8, 
     cex.axis = 0.8, ylab = 'Balota 2', xlab = 'Balota 3', col = 'red')
plot(baloto$Balota_3, baloto$Balota_4, main = "Relacion entre Balota N3 y Balota N4", cex.main = 0.8, 
    cex.axis = 0.8,  ylab = 'Balota 4', xlab = 'Balota 3', col = 'red')
plot(baloto$Balota_1, baloto$Balota_5, main = "Relacion entre Balota N1 y Balota N5", cex.main = 0.8, 
    cex.axis = 0.8,  ylab = 'Balota 5', xlab = 'Balota 1', col = 'red')
